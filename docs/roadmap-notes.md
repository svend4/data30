# Проектные заметки / Roadmap

## Быстрый обзор пользовательских кейсов и ожидаемой ценности
- **Маркетинг (рост выручки и удержание)**: сегментация аудитории, персонализация офферов и кампаний, улучшение конверсии за счёт релевантных рекомендаций.
- **Документооборот (операционная эффективность и контроль рисков)**: ускорение согласований, снижение ошибок при обработке документов, повышение прозрачности процессов.

## Критерии выбора вертикали
- **Доля целевых пользователей**: насколько большая часть клиентской базы напрямую выигрывает от решения.
- **Доступность данных**: качество, полнота и юридическая допустимость использования данных для выбранных сценариев.
- **Сложность интеграций**: количество систем, API и усилий на внедрение/поддержку.

## Выбранная вертикаль и решение
- **Вертикаль**: документооборот.
- **Решение**: фокус на автоматизации согласований и снижении ошибок обработки документов, как наиболее быстрой точке окупаемости при наличии доступных данных и ограниченной сложности интеграций.
