# План измерений для MVP

## 1. Основные воронки (Acquisition → Activation → Retention → Revenue)

| Этап | Цель | Ключевое событие/сигнал | Примечания |
| --- | --- | --- | --- |
| Acquisition | Привлечь релевантных пользователей | Первый визит/клик по кампании, регистрация | Отслеживаем каналы трафика и источники. |
| Activation | Довести пользователя до первой ценности | Завершение онбординга + ключевое действие | Определить "aha-moment" продукта. |
| Retention | Вернуть пользователя | Повторное использование ключевой функции | Фиксируем возвращаемость по когортам. |
| Revenue | Монетизировать | Оплата, апгрейд, повторная покупка | Отделяем пробные и платные события. |

## 2. Метрики и определения

| Метрика | Определение | Формула/примечания |
| --- | --- | --- |
| Конверсия по этапам | Доля пользователей, дошедших до следующего этапа | Activation/Acquisition, Retention/Activation, Revenue/Retention |
| Retention D7 | Доля пользователей, вернувшихся на 7-й день | Пользователи, активные на D7 / новые пользователи |
| Retention D30 | Доля пользователей, вернувшихся на 30-й день | Пользователи, активные на D30 / новые пользователи |
| NPS | Индекс лояльности | % промоутеров − % критиков |
| CAC | Стоимость привлечения клиента | Маркетинговые затраты / число новых клиентов |
| LTV | Жизненная ценность клиента | ARPU × средняя длительность жизни клиента |

## 3. Пороги успеха/неуспеха для MVP

| Метрика | Успех (MVP) | Риск/неуспех (MVP) |
| --- | --- | --- |
| Activation | ≥ X% | < X% |
| Retention D7 | ≥ X% | < X% |
| Retention D30 | ≥ Y% | < Y% |
| NPS | ≥ Z | < Z |
| CAC | ≤ A | > A |
| LTV | ≥ B | < B |

> Значения X/Y/Z/A/B фиксируются после первых 2–4 недель пилота и согласования с бизнесом.

## 4. Инструменты сбора данных

| Задача | Инструмент | Описание |
| --- | --- | --- |
| Продуктовая аналитика | Amplitude / Mixpanel / PostHog | События, воронки, ретеншн, когортный анализ. |
| Веб-аналитика | Google Analytics / Яндекс Метрика | Каналы привлечения и поведение на сайте. |
| Опросы и NPS | Typeform / SurveyMonkey | Регулярные опросы удовлетворенности. |
| CRM/продажи | HubSpot / Pipedrive | Воронка продаж, сделки, доход. |
| Финансовые данные | BI/ERP | Себестоимость, доход, валовая маржа. |

## 5. Шаблон дашборда и график измерений

### 5.1 Шаблон дашборда (минимальный набор)
1. **Верхнеуровневые KPI:** Acquisition → Activation → Retention → Revenue.
2. **Retention по когортам:** D1/D7/D30.
3. **Конверсия по этапам:** % переходов между этапами.
4. **NPS и комментарии пользователей.**
5. **CAC vs LTV:** соотношение и динамика.
6. **Тренды по ключевым событиям:** WAU/MAU, ключевое действие.

### 5.2 График измерений
| Периодичность | Метрики | Формат отчета |
| --- | --- | --- |
| Ежедневно | Acquisition, Activation, ключевые события | Дашборд + алерты |
| Еженедельно | Retention D7, конверсии этапов | Итоговый weekly-отчет |
| Ежемесячно | Retention D30, CAC, LTV, NPS | Business review |
