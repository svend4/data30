# Проверка каталога приложений

Этот чеклист используется перед внесением изменений в `catalog/apps.yaml` и обновлением
`catalog/compatibility.md`.

## 1) Проверка карточки приложения

- [ ] Заполнены все обязательные поля (см. `catalog/schema.md`).
- [ ] `id` уникален и не конфликтует с существующими карточками.
- [ ] `platforms` содержит только допустимые значения.
- [ ] `functions` содержит 3–7 ключевых функций.
- [ ] `compatibility` не включает собственный `id`.
- [ ] `support_status` и `maturity` соответствуют допустимым значениям.
- [ ] `licensing` заполнен полностью (model/license/cost/notes).

## 2) Проверка совместимости

- [ ] Все `compatibility` ссылаются на существующие `id`.
- [ ] Нет дублирующихся записей в списке совместимости.
- [ ] Новые интеграции отражены в `catalog/compatibility.md`.

## 3) Проверка сопоставлений функций

- [ ] Добавлены сопоставления в `docs/catalog/function-mapping-table.md`.
- [ ] Все функции существуют в `docs/function-dictionary.md`.
- [ ] Комментарии поясняют нетривиальные соответствия.
